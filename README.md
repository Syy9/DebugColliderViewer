# DebugColliderViewer
Unity Collider Viewer at runtime.

![demo](demo1.png)
![demo](demo2.png)

## Usage

```
// Show
DebugColliderViewer.Show(target);

// Hide
DebugColliderViewer.Hide(target);

// Change color
DebugColliderViewer.Color = color;
```
